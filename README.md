# ToDO Compose

A small Android To-Do app built with Jetpack Compose.

This project is currently at the starter stage, so the app is still the default Compose screen (`Hello Android!`). The goal is to grow it step by step into a clean, useful to-do app.

## Current status

- Android project is set up and running
- Compose UI theme is wired (`ToDOComposeTheme`)
- `MainActivity` and basic preview are ready
- To-do features are not implemented yet

## Tech stack

- Kotlin
- Jetpack Compose (Material 3)
- Android Studio + Gradle

## Planned next steps

- Add a task model
- Show a list of tasks
- Add create / edit / delete actions
- Save tasks locally (Room or DataStore)

## Run locally

1. Open this project in Android Studio
2. Let Gradle sync
3. Run on an emulator or connected device

## Project structure

- `app/src/main/java/com/example/todocompose` - app code
- `app/src/main/res` - resources

If you are checking this repo early: yes, it is intentionally simple right now. The core setup is done, and feature work comes next.
